## 2. 프로젝트 제품 배포 방법  
- 사용 기술
- GCP: 클라우드 인프라 제공
- Django: 웹 애플리케이션 프레임워크
- Gunicorn: WSGI HTTP 서버
- Nginx: 리버스 프록시 및 정적 파일 서버
- Let's Encrypt: 무료 SSL 인증서 제공

배포 절차

2. 루트 디렉토리에 레포지토리 클론
